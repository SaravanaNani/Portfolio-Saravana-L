## 🚀 Getting Started

Clone the repo:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```
### Install dependencies:
```bash
npm install
```
### Run locally:
```bash
npm run dev
```
### Build for production:

```bash
npm run build
npm start

```
